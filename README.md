# VoltaPraMim API

Backend do sistema VoltaPraMim, uma plataforma de achados e perdidos para universidades.

## Tecnologias

- Node.js
- Fastify
- TypeScript
- Prisma
- PostgreSQL
- JWT

## Funcionalidades

- Registro de usuário
- Login com autenticação JWT
- Criar item encontrado
- Listar itens
- Buscar item por ID
- Editar item

## Como rodar o projeto

Instalar dependências

npm install

Rodar servidor

npm run dev

Servidor roda em

http://localhost:3333
